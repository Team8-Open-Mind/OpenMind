# 💻 프로젝트 실행시키기
1. 프로젝트를 clone 하세요.
   ```
   git clone https://github.com/your_username_/Project-Name.git](https://github.com/Team8-Open-Mind/OpenMind.git
   ```
3. NPM 패키지들을 설치하세요.
   ```
   npm install
   ```

   ```js
   npm run dev
   ```
